# Neovim Keybindings

Leader key: `Space`

## Telescope (find)

| Key | Action |
|-----|--------|
| `<leader>ff` | Find files |
| `<leader>fg` | Live grep |
| `<leader>fb` | Buffers |
| `<leader>fh` | Help tags |

## LSP

| Key | Action |
|-----|--------|
| `gd` | Go to definition |
| `K` | Hover docs |
| `<leader>rn` | Rename symbol |
| `<leader>ca` | Code action |
| `gr` | References |
| `<leader>cf` | Format file |

## File Explorer

| Key | Action |
|-----|--------|
| `<leader>e` | Toggle Neo-tree |

## Completion (insert mode)

| Key | Action |
|-----|--------|
| `<C-Space>` | Trigger completion |
| `<Tab>` | Next item / expand snippet |
| `<S-Tab>` | Prev item / jump back snippet |
| `<CR>` | Confirm selection |
| `<C-b>` | Scroll docs up |
| `<C-f>` | Scroll docs down |
| `<C-e>` | Abort completion |

## Copilot (insert mode)

| Key | Action |
|-----|--------|
| `<M-l>` | Accept suggestion |
| `<M-]>` | Next suggestion |
| `<M-[>` | Prev suggestion |
| `<M-e>` | Dismiss suggestion |

## Debugging

| Key | Action |
|-----|--------|
| `<F5>` | Continue |
| `<F9>` | Toggle breakpoint |
| `<F10>` | Step over |
| `<F11>` | Step into |

## Telescope Picker (insert mode)

| Key | Action |
|-----|--------|
| `<C-j>` | Next result |
| `<C-k>` | Prev result |
